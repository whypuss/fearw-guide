Telegram 接入

FearW Gateway 支持 Telegram。配置完成後，可以直接在 Telegram 裡和 FearW 對話。

1. 創建 Telegram Bot

1

找 @BotFather

在 Telegram 搜索 @BotFather，發送：

/newbot
2

設定 Bot 名稱

按提示輸入 Bot 名稱和 username。

BotFather 會給你一個 Bot Token，格式如：123456789:ABCdefGHIjklMNOpqrsTUVwxyz

這個 Token 是敏感資料，請妥善保管，不要上傳到 GitHub。

2. 配置 FearW Telegram Gateway

在 Profile 配置中加入 Telegram

編輯 ~/.hermes/profiles/fearw/config.yaml：

platforms: telegram: enabled: true bot_token: "你的BotToken" admin_ids: - 你的TelegramUserID # 管理員，可以執行 /model 等命令 allowed_chats: [] # 空 = 所有人均可使用；或填寫允許的 chat_id

如何獲得自己的 Telegram User ID？

搜索 @userinfobot 或 @getidsbot，發送任意消息，它會返回你的 User ID。

3. 啟動 FearW 並開啟 Gateway

完整啟動命令

cd ~/.hermes/hermes-agent-fearw && source .venv/bin/activate hermes -p fearw gateway run

這會啟動 FearW Gateway，監聽所有配置好的平台（包括 Telegram）。

只測試 Telegram（不啟動其他平台）

可以在 config.yaml 裡只啟用 Telegram，其他平台維持關閉。

4. 在 Telegram 裡測試

  1. 在 Telegram 找到你的 Bot（用 username 搜索）
  2. 發送 /start
  3. 發送 /help 查看可用命令
  4. 直接發送消息開始對話

5. 常用 / 命令

命令說明
/start開始對話
/help顯示幫助
/model切換模型
/sessions查看 / 管理對話歷史
/tools查看可用工具
/memory查看記憶狀態

6. 進階配置

限定只有自己能使用

platforms: telegram: enabled: true bot_token: "你的BotToken" admin_ids: - 你的TelegramUserID allowed_chats: - 你的TelegramChatID # 群組的 chat_id，或 DM 的 chat_id

開機自動啟動（macOS launchd）

創建 ~/Library/LaunchAgents/com.hermes.fearw.plist：

<?xml version="1.0" encoding="UTF-8"?> <plist version="1.0"> <dict> <key>Label</key><string>com.hermes.fearw</string> <key>ProgramArguments</key> <array> <string>/Users/你的用戶名/.hermes/hermes-agent-fearw/.venv/bin/hermes</string> <string>-p</string> <string>fearw</string> <string>gateway</string> <string>run</string> </array> <key>RunAtLoad</key><true/> <key>WorkingDirectory</key><string>/Users/你的用戶名/.hermes/hermes-agent-fearw</string> </dict> </plist>

請把 你的用戶名 替換為實際的 macOS 用戶名。
